# Project Instructions for AI Agents

## UI & Aesthetics

### Color Scheme
When generating or modifying UI components, STRICTLY adhere to the following color palette defined in `globals.css` as Tailwind variables:

- **Accent Yellow** (`--color-accent`: `#ecad0a`): Use via `bg-accent`, `text-accent`, etc. Intended for accent lines and highlights.
- **Blue Primary** (`--color-primary`: `#209dd7`): Use via `bg-primary`, `text-primary`. Intended for links, active states, and key sections.
- **Purple Secondary** (`--color-secondary`: `#753991`): Use via `bg-secondary`, `text-secondary`. Intended for submit buttons and critical actions.
- **Dark Navy** (`--color-navy`: `#032147`): Use via `bg-navy`, `text-navy`. Intended for main headings and strong copy.
- **Gray Text** (`--color-graytext`: `#888888`): Use via `text-graytext`. Intended for supporting text, descriptions, and labels.

**Backgrounds:** Use `bg-slate-50` or `bg-white` instead of generic dark themes, complementing the Dark Navy text.
